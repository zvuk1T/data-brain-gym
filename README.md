## Core rule

> I often know more than I need to say.  
> The goal is to answer clearly, calmly, and briefly вЂ” not to explain everything at once.

## How I train

```text
1. Solve a small problem
2. Explain the solution
3. Review what was hard
4. Repeat consistently
```
